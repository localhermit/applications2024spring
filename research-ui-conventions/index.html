    This element has been used by other sites/apps when there are a lot of variations or options for the user to choose from.

    It was intuitive to use because there was a subtitle saying "Filters". Additionally, it is also a very common element seen on a lot of sites/apps, so for those who have been on the internet for a while it is easy to know how to use the element.

User Story Analaysis

google sheet

Each user story is categorized by action and goal- what is the goal of the user and what action do they need to do. In addition, the user stories are scored based on the priority of the action/goal to the user, how common the needs were, and how achievable were each goal on a scale of 1-3 (3- very important and 1- not very important). The mean of the three scores were taken to determine the importance of each user story.

I then created 4 graphs

Graph 1: each persona to goal category to find out which was the most important to each persona.

Graph 2: which goal was scored the highest amongst all the personas

Graph 3: each persona to action category to find out which was the most important to each persona.

Graph 4: which action was scored the highest amongst all the personas

Result 1:

BMXer -> portability, spec, speed

family senior -> spec + budget

middle age -> budget + maintenance

parent + child -> affordability + budget

teen -> spec, adaptability, durability, optimization, purpose

Result 2: maintenance, optimization, spec

Result 3:

BMXer -> durability

fitness + family senior -> height + stability

teen -> spec, budget, durability, review, policies

Result 4: aesthetic, price, maintenance, spec, speed, weight
